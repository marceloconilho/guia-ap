# Apêndice — Mapas de compressor

Este apêndice reúne os mapas reais disponíveis em fontes públicas e explica como eles serão usados no manual.

## Aviso importante

Os mapas oficiais podem ter direitos autorais dos fabricantes. Por isso, nesta base nós:

- linkamos as fontes oficiais;
- usamos desenhos esquemáticos para ensinar leitura;
- não redesenhamos mapas proprietários como se fossem nossos;
- marcamos claramente o que é mapa real e o que é diagrama didático.

## Como ler

## Garrett G25-550

- Fonte oficial: https://www.garrettmotion.com/racing-and-performance/performance-catalog/turbo/g-series-g25-550/
- Dados principais: 300-550 hp, 1.4L-3.0L, compressor 48 mm, eficiência máxima publicada de 80%, velocidade máxima de 185 krpm.
- Mapa real: disponível no site oficial Garrett, seção "Compressor Map".

## Garrett G25-660

- Fonte oficial: https://www.garrettmotion.com/racing-and-performance/performance-catalog/turbo/g-series-g25-660/
- Dados principais: 350-660 hp, 1.4L-3.0L, compressor 54x67 mm segundo sell sheet oficial.
- Mapa real: disponível no site oficial Garrett, seção "Compressor Map".

## BorgWarner EFR 6258

- Fonte oficial: https://www.borgwarner.com/docs/default-source/iam/boosting-technologies/efr-6258-a.pdf?sfvrsn=595bb03c_17
- Dados principais: 225-450 hp, compressor 49x62 mm, turbina 51x58 mm, Gamma-Ti, dual ceramic ball bearing, FMW.
- Mapa real: aparece no PDF oficial.

## Master Power R4449

- Fonte de dimensões: https://www.regisracing.com.br/turbinas/master-power-turbo-r4449-2-145-360-hp-44-05-x-49-5
- Dados principais: compressor 44,05x63 mm, turbina 49,5x61,5 mm, A/R quente .48 mono, faixa 145-360 hp.
- Observação: mapas completos públicos são mais difíceis de localizar em fonte oficial atual. Quando não houver mapa oficial, o manual usa faixa declarada + dimensões + experiência prática, sempre sinalizando a limitação.

## Auto Avionics A50-2

- A50-2.36M oficial: https://autoavionics.com.br/produto/a50-2-36m/
- A50-2.48P oficial: https://autoavionics.com.br/produto/a50-2-48p/
- Dados principais: compressor 51x71 mm, turbina 49,4x58 mm, versões monofluxo e pulsativa.
- Observação: até esta base, não foi identificado mapa de compressor público oficial completo como os da Garrett/BorgWarner.
